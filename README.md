##更新日志：
[v0.1.0] - 20260509
项目完成初始化，已能够稳定运行。
第一版功能如下：
1. 在 Windows 本机稳定启动。
2. 显示透明背景的人物。
3. 测试播放 idle、waving、running、failed、waiting、review 等基础动作。
4. 用户点击人物后能输入文字进行交流。
5. 成功调用 DeepSeek 官方 API 并显示回复。
6. 回复符合可爱、温柔、简短、自然的人格。
7. 启动时主动问候。
8. 空闲时静默显示主动气泡。
9. 用户不回复时，20 分钟内不会重复主动打扰。
10. 免打扰模式可关闭主动行为。
11. 聊天记录可保存为 JSON，并可以根据对话自动加载。
12. 上下文过长时可进行自动摘要。
13. API 失败、网络失败时程序不崩溃。
14. 记住上次退出位置,并在下次启动时自动加载。
